# School Facilities Complaint System

A web-based complaint management system designed to help students report damaged school facilities efficiently.

## Features

- Student complaint submission
- Complaint status tracking
- Category management
- Student management
- Admin dashboard
- Complaint update system

## Technologies Used

- PHP
- MySQL
- Bootstrap
- JavaScript
- HTML
- CSS
